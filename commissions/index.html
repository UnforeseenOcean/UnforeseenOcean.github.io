What is this?

This is a free service for designing your own websites.

Please read the entire page before requesting!

Service features:

  • A complete, singular webpage (NOT whole website)
  • Free QA service
  • Free tech support
  • Ideal for landing pages
  • Good for basing your own websites on
  • Minimal files to upload
  • Lightweight page
  • Minimal amount of JavaScript used -- Speeds up the page
  • Minimal CDN (Content Distribution Network) reliance -- More reliable page, easy to maintain
  • Easy to modify, throughly documented code

However, this service does have caveats:

  • The page is static - No dynamic content support out-of-the-box.
  • The page cannot be used for sites such as WordPress, Tumblr, BlogSpot, Squarespace or Wix, as they're not compatible.
  • The page will be simple. No fancy designs will be done.
  • The page cannot be updated without changing the code.
  • No fancy animations are possible due to the simple JavaScript code.
  • The page only uses vanilla JavaScript. This will not use JavaScript frameworks such as Vue.js, Bootstrap or jQuery.
  • The process will be slow, as it will be processed whenever I have free time.
What do I need?

You'll need to provide the following:

  • Info on what the page will be used for
  • A sketch of the page's layout
  • Any logo or content that'll be included in the page
  • That's all!
