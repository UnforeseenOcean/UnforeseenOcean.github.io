Ascender Readme

Ascender is a server replacement for the game Ascend: Hand of Kul, which has shut down. This is more of a proof-of-concept, so expect bugs to occur. And no, we are not the original developers of the game, and we are not in any way endorsed or affiliated with Signal Studios.

Installation

The NEW and IMPROVED launcher should do it automatically for you, which you can get through our Discord server: Discord server

Should the launcher not install the game for you, click on the link to start the game on Steam: Install Ascend: Hand of Kul on Steam

Don't have Steam installed? You should have it installed first! Go install it now.

Playing the game

Simply start the launcher you downloaded from our Discord server. Enjoy!

IMPORTANT: Do not start the game from Steam. This is the launcher you should use, unless specifically required.

Common issues

  • Crashes on some AMD Ryzen processors

    A bug in the game engine causes the game to crash. Download the patched version of the game from our Discord server and put it in the game's bin folder.

  • XINPUT1_3.dll and d3dx9_43.dll error

    Please install DirectX 9 Runtime from Microsoft. This does not come pre-installed on Windows 10.

  • "Ascend Service Not Available"

    Use the launcher to start the game. NOT from Steam.

  • Game crashes or black-screens when Alt-tabbing

    Use Windowed mode. (Click Play on Steam, click on Use Launcher, set the window mode to Windowed, click Launch then close the game)

  • Where did the launcher go? I'm sure I downloaded it...

    Your antivirus program (which may include your built-in Windows Defender) is deleting the file due to a false positive match. Add the launcher executable to the exception list, disable the antivirus while playing or use other antivirus programs.

  • Invalid Resolution error

    This may happen when you first start the game. Click Play on Steam, click on Use Launcher, set the desired resolution, click Launch, close the game and try again.

  • The launcher does not start

    Unpack the full archive (consult G**gle or other search engines if you have no idea how to do so) into a folder then try again.

  • Game does not work on Windows 10 despite trying everything

    Try reinstalling the game outside of the default folder. This is not a recommended fix, but it sometimes resolves the issue on Windows 10 systems. Before you try this, try other solutions first.
